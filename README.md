## 🎯 Objetivo
Realizar uma **Análise Exploratória de Dados (EDA)** do dataset *Brazilian Cities*, buscando identificar padrões socioeconômicos, demográficos e de infraestrutura, com foco em **destinos turísticos com potencial ainda pouco explorado**.

---

## 📂 Dataset
- Fonte: Kaggle  
- Link: https://www.kaggle.com/datasets/crisparada/brazilian-cities  

---

## 🚀 Sobre o Projeto
O Brasil possui mais de **5.500 municípios** com realidades diversas.  
Este projeto tem como objetivo analisar esses dados para identificar **cidades com bons indicadores, mas baixa visibilidade turística**, contribuindo para insights no contexto do aplicativo *Brasil em Foco*.

---

## 🔍 Etapas da Análise

1. **Carregamento dos dados**  
2. **Limpeza e tratamento**  
3. **Análise univariada**  
4. **Análise bivariada e multivariada**  
5. **Identificação de outliers**  
6. **Geração de insights**
